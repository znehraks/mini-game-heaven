# Design System Creation Prompt

## Context
You are designing the UI/UX for "Mini-Game Heaven" - a web arcade with mini games targeting Korean teenagers (10대).

## Design Direction
- **Aesthetic**: Vaporwave / Y2K / Retro Arcade
- **Mood**: Nostalgic, fun, competitive, addictive
- **Target**: Mobile-first, 10대 Korean teenagers

## Confirmed Colors (from tech_stack.md)
- Neon Pink: #ff00ff
- Neon Cyan: #00ffff
- Neon Purple: #8b5cf6
- Arcade Dark: #0a0a0f (background)
- Arcade Darker: #050508 (deeper background)

## Confirmed Font
- Headings: 'Press Start 2P' (pixel/arcade font)
- Body: System UI / Sans-serif

## Task
Create a comprehensive design system including:

### 1. Extended Color Palette
- Primary, secondary, accent colors
- Gradient definitions
- Glow/neon effects
- Text colors (primary, secondary, muted)
- Status colors (success, warning, error)
- Surface colors (cards, modals)

### 2. Typography Scale
- Heading sizes (h1-h6)
- Body text sizes
- Game UI text
- Score display typography

### 3. Spacing System
- Base unit
- Spacing scale (xs, sm, md, lg, xl, 2xl)
- Component padding
- Section margins

### 4. Border & Effects
- Border radius scale
- Neon glow effects (CSS)
- Shadow definitions
- Animation timings

### 5. Breakpoints
- Mobile-first breakpoints
- Game canvas sizes

### 6. Component Tokens
- Button styles (primary, secondary, ghost)
- Card styles
- Input styles
- Modal styles

Output as a structured design_system.md file with Tailwind CSS configuration examples.
